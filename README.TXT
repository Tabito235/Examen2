Este software funciona mediante SpringBoot, una base de datos hecha en SQL y JDBC, siendo un CRUD en el cual podemos
ingresar datos mediante la tabla principal y posteriormente ir modificando o eliminando los datos, la primera pestaña
es meramente para poder ingresar todo mas sencillo, solo dandole a iniciar sesion, y de hay puedes usar las funcioones de la 
tabla para poder añadir, editar, visualziar o eliminar los datos.
